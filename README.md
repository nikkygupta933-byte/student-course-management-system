# Student Course Management System

A full-stack Student Course Management System built using Java, Spring Boot, Thymeleaf, Bootstrap, and MySQL. The application helps manage student records through a user-friendly interface with complete CRUD operations.

---

## 🚀 Features

- Add Student Records
- Update Student Information
- Delete Student Records
- View Student Details
- Manage Course Information
- Responsive User Interface
- Form Validation
- CRUD Operations
- MySQL Database Integration

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/nikkygupta933-byte/student-course-management-system.git
```

2. Open the project in IntelliJ IDEA.

3. Configure MySQL database in `application.properties`.

4. Run the Spring Boot application.

5. Open your browser and access the application.
